# Dockerized Cuckoo
Dockerized Cuckoo sandbox for distributed usage. Some functionality modified/omitted  

# Installation
1. Install Virtualbox
2. Configure VirtualBox Web service
3. Change configs in `./config` folder
4. Build and launch
```
docker-compose up --build
```
